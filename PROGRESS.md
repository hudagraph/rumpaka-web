# 📊 Progres Migrasi Next.js — PT Sanggar Seni Annisa Rumpaka

## ✅ Pekerjaan Selesai
1.  **Inisialisasi Project**: Next.js 16 (App Router) dengan TypeScript.
2.  **Sistem Desain**: Migrasi variabel CSS, font, dan reset style ke `globals.css`.
3.  **Komponen Global**:
    *   `Navbar.tsx`: Navigasi responsif dengan efek scroll dan menu mobile.
    *   `Footer.tsx`: Layout grid multi-kolom.
    *   `CustomCursor.tsx`: Kursor interaktif premium menggunakan Framer Motion.
4.  **Halaman Utama (Home)**: Hero slider dengan animasi transisi smooth dan swipe logic.
5.  **Halaman Layanan**: Layout alternatif dengan reveal animation saat scroll.
6.  **Halaman Kontak**: Form interaktif dan info kartu kontak.
7.  **Halaman Arum Kostum**: Katalog dengan filter kategori dan sistem modal detail.
8.  **Halaman Art School**: Program belajar, kurikulum stages, dan tim pengajar.
